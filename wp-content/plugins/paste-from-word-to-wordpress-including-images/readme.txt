Allows pasting from Word to WordPress when combined with NonprofitCMS.org Word to WordPress tool

== Description ==

The NonprofitCMS.org paste from Word to WordPress allows you to copy a Microsoft Word document and paste it into your WordPress visual editor.

The plugin automatically uploads images and charts.

Microsoft Windows XP or greater is required along with the NonprofitCMS.org paste from Word to WordPress desktop widget

Visit plugin website <http://www.nonprofitcms.org/2011/03/paste-from-word-to-wordpress-including-images/> for Microsoft Windows based widget to enable uploadability



== Installation ==
Download: [Paste from Word to WordPress](http://www.nonprofitcms.org/2011/03/paste-from-word-to-wordpress-including-images/ "Paste from Word to WordPress")

1. Install the plugin via WordPress.org or unzip the contents of the file into your wp-content/plugins folder.

2. Activate the plugin

3. Run Setup.exe file on your Microsoft Windows Machine

4. Enter your website URL + security activation code

5. Copy a Word document

6. Paste it into WordPress!
